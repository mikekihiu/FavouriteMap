FAQs

How to bookmark a location

  • Navigate to the Home screen
  • Seach for the location
  • Select the location from the results list
  • Navigate back to the Home screen
  • Dismiss search
  • The location is now on the list of bookmarked locations

How to delete a bookmarked location

  • Navigate to the Home screen
  • Swipe left on a bookmarked location
  • Select Delete

How to view the weather forecast

  • Enter your openweather api key at WeatherAPI.swift file if you haven't already done so
  • Navigate to the Home screen
  • Tap a bookmarked location

How to change weather unit system

  • Navigate to the Home screen
  • Tap the top left icon
  • Select Settings
  • Tap unit sytem
  • Select the preferred unit sytem from the picker
