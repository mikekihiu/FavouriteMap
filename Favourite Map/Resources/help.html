FAQs

How to bookmark a location

  • Navigate to the Home screen
  • Search for the location
  • Select it from the results list

How to delete a bookmarked location

  • Navigate to the Home screen
  • Swipe left on a bookmarked location
  • Select Delete

How to view the weather forecast

  • Navigate to the Home screen
  • Tap a bookmarked location
  • Alternatively, bookmark a location, then tap the chevron up button

How to change weather unit system

  • Navigate to the Home screen
  • Tap the top left icon
  • Select Settings
  • Tap unit sytem
  • Select the preferred unit sytem from the picker
